Data for Bachelor Thesis
Title: Optimizacija rezanja pločastih materijala s usporedbom algoritama i analizom učinkovitosti

This repository contains raw execution logs and statistical analysis data supporting the findings presented in Chapter 5.8 of the thesis.
raw-logs: Individual execution traces for 150 test runs (50 per optimization profile).
analysis: Aggregated data, distribution calculations, and source data for graphs.
